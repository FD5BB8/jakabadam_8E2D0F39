Fooldal Build Particle System Texture Handling & Animáció

Udvozollek a Unity bemutato oldalon!

Ez a weboldal roviden bemutatja a Unity jatekmotor nehany fontos funkciójat.

  • Particle System – Hogyan lehet vizualis effekteket, mint peldaul tuzet vagy fustot letrehozni.
  • Texture Handling – Hogyan kezelhetjuk a kulonbozo texturakat (Diffuse, Normal Map, Specular, stb.).
  • Animation Framework – Hogyan mukodnek az animáciok, FPS és frame számítás.
  • Build Check – Hogyan tudjuk ellenorizni a celplatformot (PC vs. Mobil).

A weboldal nemcsak leirasokat tartalmaz, hanem interaktiv JavaScript peldakat is, amelyek Unity-hez kapcsolodo szamitasokat vegeznek.
